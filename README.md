# 1 前言

这次的可视化大屏是2021年春节做的一个项目的demo。

# 2 简介

本项目主要基于百度的Echarts框架实现数据的可视化，主要包括饼状图、柱状图、折线图、信息轮播等，背景为黑色，并且css的样式中做了三个宽度的样式做到对宽度的自适应。
